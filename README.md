# QUL Ayah Explorer

A React app for exploring Quran ayahs with Arabic text, translations and audio — built on **Tarteel's [Quranic Universal Library (QUL)](https://qul.tarteel.ai)**.

## QUL Resources Used
| Resource | QUL Category |
|---|---|
| Arabic text (Uthmani script) | Quran Scripts |
| English translations | Translations |
| Audio recitations | Recitations and segments data |
| Surah metadata | Quran metadata |

## Getting Started
```bash
npm install
npm run dev
```
